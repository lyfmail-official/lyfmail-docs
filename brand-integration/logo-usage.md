# Logo Usage

## Primary Logo

The LYF Mail logo consists of a circular icon with an abstract "L" shape and the wordmark "LYF Mail".

## Correct Usage

- Maintain clear space: minimum 1x logo height on all sides.
- Use on light or dark backgrounds with appropriate color variant.
- Scale proportionally — never stretch or distort.

## Incorrect Usage

- Do not rotate the logo.
- Do not change the colors outside approved palette.
- Do not add effects (shadows, glows, 3D).
- Do not use the icon without the wordmark (except at very small sizes).

## Minimum Sizes

| Context | Minimum Width |
|---------|--------------|
| Digital | 120px |
| Print | 40mm |
| Favicon | 32px |
