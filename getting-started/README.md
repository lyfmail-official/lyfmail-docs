# Getting Started with LYF Mail

Welcome to the LYF Mail ecosystem! This section will help you understand the platform and deploy your first project quickly.

## What Is LYF Mail?

[LYF Mail](https://lyfmail.com) is a privacy-first digital wellness platform serving 50,000+ subscribers across India since 2020. It delivers curated newsletters and free open-source tools for health, wealth, career, creativity, and mindfulness.

## The LYF Mail Ecosystem

- **Main Website:** [lyfmail.com](https://lyfmail.com) — Newsletters and guides
- **PWA:** [app.lyfmail.com](https://app.lyfmail.com) — Progressive Web App
- **Docs:** [docs.lyfmail.com](https://docs.lyfmail.com) — This documentation hub
- **Tools:** 9 free, open-source privacy tools across multiple subdomains
