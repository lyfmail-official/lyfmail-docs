## Key Principles

1. **One subdomain = one topic focus** — Clear topical authority per subdomain.
2. **Cross-link naturally** — Link between related tools and content.
3. **Schema markup everywhere** — Rich snippets for all content types.
4. **Fast performance** — Core Web Vitals optimization across all properties.
5. **Mobile-first** — All documentation and tools optimized for mobile.
