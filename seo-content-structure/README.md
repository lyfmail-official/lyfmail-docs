# SEO & Content Structure

Learn how LYF Mail optimizes its multi-subdomain ecosystem for search engines and organic discovery.

## Topics

- Indexing Strategy — Content structure for maximum search visibility.
- Sitemap Configuration — XML sitemaps for multi-subdomain sites.
- Internal Linking — Cross-linking between lyfmail.com and subdomains.
- Schema.org Markup — Structured data for articles, FAQs, and apps.
- Subdomain SEO — Best practices for targeted content discovery.

## Key Principles

1. **One subdomain = one topic focus** — Clear topical authority per subdomain.
2. **Cross-link naturally** — Link between related tools and content.
3. **Schema markup everywhere** — Rich snippets for all content types.
4. **Fast performance** — Core Web Vitals optimization across all properties.
5. **Mobile-first** — All documentation and tools optimized for mobile.
