# PWA Setup Guide

Learn how to build, configure, and deploy Progressive Web Apps within the LYF Mail ecosystem.

## Topics

- File Structure — Standard LYF Mail PWA directory layout.
- Manifest Configuration — `manifest.json` best practices.
- Service Worker — Caching strategies and offline support.
- Offline Mode — Designing for low-connectivity environments.

## Principles

All LYF Mail PWAs follow these core principles:

- **Privacy-first** — No tracking, no analytics, minimal permissions.
- **Offline-first** — Core functionality works without internet.
- **Lightweight** — Fast load times, minimal dependencies.
- **Accessible** — WCAG 2.1 AA compliant.
