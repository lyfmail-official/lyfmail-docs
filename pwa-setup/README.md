## Principles

All LYF Mail PWAs follow these core principles:

- **Privacy-first** — No tracking, no analytics, minimal permissions.
- **Offline-first** — Core functionality works without internet.
- **Lightweight** — Fast load times, minimal dependencies.
- **Accessible** — WCAG 2.1 AA compliant.
